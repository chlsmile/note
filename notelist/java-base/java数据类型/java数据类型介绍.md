## java数据类型介绍

### 简介

> Java是一门强类型语言,也就是说,所有的变量都必须显式声明类型。Java的数据类型分为两大类:原始类型(primitive type,也称为简单类型)和引用类型(reference type)


### 数据类型的作用(数据类型约束了两件事情)

- 能存储的信息是什么样的格式;
- 能存储的信息的范围是多少.
