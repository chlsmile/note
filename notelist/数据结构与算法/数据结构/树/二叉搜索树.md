## 二叉搜索树(Binary Search Tree)

#### 二叉搜索树的特点
- 二叉搜索树必须是一棵二叉树;
- 若它的左子树不空，则左子树上所有结点的值均小于它的根结点的值;
- 若它的右子树不空，则右子树上所有结点的值均大于它的根结点的值;


#### 二叉搜索树常用操作实现
```java
public class BinarySearchTree {

    /**根节点*/
    public Node root;
    

    /**
     * Node 树节点
     */
    class Node{

        /**节点数据*/
        public int data;

        /**左孩子*/
        public Node leftChild;

        /**右孩子*/
        public Node rightChild;

        public Node(int data) {
            this.data = data;
        }

    }
    
    

    /**
     * find 二叉搜索树查找指定的节点
     *
     * 查找流程，采用递归的思路，从根节点开始
     *
     * 如果要查找的值data与当前节点的data值相同则直接返回当前节点
     *
     * 如果要查找的值data小于当前节点，则将当前节点的左子树赋值给当前节点，继续进行while循环
     *
     * 如果要查找的值data大于当前节点，则将当前节点的右子树赋值给当前节点，继续进行while循环
     *
     * 如果当前节点为空表示要查询的节点不存在，返回null
     *
     *
     * @param data 要查找的节点的数据值
     *
     * @return 查找到的节点，如果树为空或者节点不存在则返回null
     */
    public Node find(int data){

        Node current = root;
        while (current!=null){
            if(data==current.data){
                return current;
            }
            if(data<current.data){
                current=current.leftChild;
            }else {
                current=current.rightChild;
            }
        }

        return null;
    }
    
    

    /**
     * findMin 二叉查找树获取最小节点
     *
     * 查找流程，采用递归的思路，从根节点开始
     *
     * 找最小值，先找根的左节点，然后一直递归找这个左节点的左节点，直到找到没有左节点的节点，那么这个节点就是最小值
     *
     *
     * @return 树节点最小值，如果树为空则返回null
     */
    public Node findMin(){

        Node current=root;
        while (current!=null){
            Node leftChild=current.leftChild;
            if(leftChild==null){
                return current;
            }
        }

        return null;
    }
    
    

    /**
     * findMax 二叉查找树获取最大节点
     *
     * 查找流程，采用递归的思路，从根节点开始
     *
     * 找最大值，先找根的右节点，然后一直递归找这个右节点的右节点，直到找到没有右节点的节点，那么这个节点就是最大值
     *
     * @return 树的节点最大值，如果树为空则返回null
     */
    public Node findMax(){

        Node current=root;
        while (current!=null){
            Node right=current.rightChild;
            if(right==null){
                return current;
            }
        }

        return null;
    }
```
